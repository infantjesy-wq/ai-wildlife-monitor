# 🦒 AI Wildlife Monitor

An AI-powered system that detects animals in images/videos and sends real-time WhatsApp alerts.

---

## 🚀 Project Overview

AI Wildlife Monitor is a Computer Vision application built using YOLOv8 that identifies animals and triggers alerts.  
This project demonstrates how to move from building ML models to deploying real-world AI systems.

---

## 🔍 Features

- 🧠 Animal detection using YOLOv8  
- 🖼️ Image-based detection  
- 🎥 Video detection with highlighted bounding boxes  
- 📱 Real-time WhatsApp alerts using Twilio API  
- 🌐 Simple UI built with Streamlit  

---

## 🛠️ Tech Stack

- Python  
- YOLOv8 (Ultralytics)  
- OpenCV  
- Streamlit  
- Twilio API  

---
## ⚙️ Installation & Run Locally

### 1. Clone the repository

```
git clone https://github.com/infantjesy-wq/ai-wildlife-monitor.git
cd ai-wildlife-monitor
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the app

```
streamlit run app.py
```

## 📸 How It Works

1. Upload an image or video  
2. Model detects animals using YOLOv8  
3. Detected objects are highlighted  
4. Alert is sent via WhatsApp
